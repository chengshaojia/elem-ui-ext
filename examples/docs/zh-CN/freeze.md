## Freeze 冻结窗格

通过鼠标或键盘输入字符

### 基础用法

::: demo
```html
<format-number v-model="input" placeholder="请输入内容"></format-number>

<script>
export default {
  data() {
    return {
      input: ''
    }
  }
}
</script>
```
:::
